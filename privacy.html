プライバシーポリシー

最終更新日: 2026年6月20日

本プライバシーポリシー（以下「本ポリシー」）は、Sohei Fujita（以下「開発者」）が提供する iOS アプリケーション ClipsForTwitch（以下「本アプリ」）における、 ユーザー情報の取り扱いについて定めるものです。

1. 取得する情報

本アプリは、以下の情報を取得します。

情報 取得タイミング 取得元
Twitch ユーザー ID / 表示名 / プロフィール画像 URL Twitch アカウントでログインした時のみ Twitch Helix API
アクセストークン / リフレッシュトークン Twitch アカウントでログインした時のみ Twitch OAuth サーバー
ユーザーが登録した配信者リスト・お気に入りクリップ アプリ内操作時 ユーザー入力

Twitch アカウントでログインしない場合（ゲスト利用時）、上記のうち Twitch 関連情報は一切取得しません。

2. 利用目的

  • Twitch API を通じたクリップ情報の取得・表示
  • Twitch API のレート制限を分散するためのユーザー識別
  • ユーザーが登録した配信者・お気に入りの保持と表示

3. 情報の保管場所

情報保管場所iCloud 同期
アクセストークン / リフレッシュトークン iOS Keychain（端末ローカル） なし
Twitch ユーザー ID / 表示名 / プロフィール画像 URL SwiftData（端末ローカル） なし
配信者リスト・お気に入りクリップ SwiftData（端末ローカル） なし

本アプリは、これらの情報を開発者のサーバーや外部サービスへ送信しません （Twitch API の呼び出しを除く）。

4. 第三者提供

本アプリは、ユーザー情報を以下を除き第三者に提供しません。

  • Twitch Interactive, Inc. — クリップ情報の取得および OAuth 認証のために、Twitch Helix API を呼び出します。 Twitch のプライバシーポリシーは こちら をご参照ください。

5. Cookie / トラッキング / 広告

本アプリは、解析ツール（Google Analytics 等）および広告 SDK を使用していません。 ユーザーの行動ログを開発者が収集することはありません。

6. データの削除

  • アプリ内の「ログアウト」操作により、Keychain 内のトークンを削除します。
  • アプリを端末からアンインストールすると、保存された全データ（配信者リスト・お気に入り・トークン）が削除されます。
  • Twitch 側の連携解除は Twitch の Connections 設定 から行えます。

7. 子どもの個人情報

本アプリは 13 歳未満（米国 COPPA 基準）または日本国内の未成年者を対象としていません。 保護者の同意なく未成年者が本アプリを利用していることが判明した場合、関連データを速やかに削除します。

8. セキュリティ

アクセストークンは iOS Keychain の kSecAttrAccessibleAfterFirstUnlockThisDeviceOnly 属性で保管され、 端末ロック解除後のみ復号可能かつ iCloud 同期対象外です。 Twitch API との通信はすべて HTTPS で暗号化されています。

9. 本ポリシーの変更

本ポリシーは、法令の変更や本アプリの機能追加に伴い改定されることがあります。 重要な変更がある場合は、本アプリまたは本ページで通知します。 最新版は常に本ページで公開します。

10. 準拠法

本ポリシーは、日本国の法令（個人情報保護法を含む）および Apple App Store Review Guidelines に準拠して運用されます。
